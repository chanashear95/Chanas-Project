Card Matching Memory Game

To start the game you must click on two cards to flip them over. If the cards are a macth they will disappear. If they are not a match they will flip over after one second. You must try to remember where each card is in order to find matches. If you find all the matches by the end of the buzzer, you win! Choosing a game difficulty starts the buzzer.
